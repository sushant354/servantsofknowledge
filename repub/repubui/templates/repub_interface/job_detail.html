Review Pages Mode
What happens when you click "Review Pages"?
  • Job Status Change: The job will be put into review mode
  • New Directory: A new review directory will be generated with cropped page images
  • Page Editor: You'll be able to adjust crop boundaries for individual pages
  • Reprocessing: After making changes, you can finalize to generate the final PDF

Retry Job
Are you sure you want to retry this job?
  • Fresh Processing: The job will be reprocessed from scratch with the same settings
  • Status Reset: Job status will change to processing
  • File Replacement: Previous output files will be replaced with new results
  • Time Required: Processing may take several minutes depending on document size

Stop Job Processing
Important: How "Stop" Actually Works
  • Background Process: The actual processing cannot be immediately stopped
  • Job Status Change: The job will be marked as failed
  • Processing Continues: Background processing may continue for a short time
  • Results Lost: Any partial progress will be discarded
After Stopping, You Can:
  • Retry the job with the same settings
  • Modify processing parameters and resubmit
  • Access input files to process elsewhere
  • Delete the job if no longer needed

Derive Job
What happens when you derive a job?
  • Directory Creation: A new directory will be created using the Identifier from metadata
  • Input File Copy: The original input file will be copied to the derive directory
  • Output Images: All output images will be zipped and copied to the derive directory
  • Thumbnails: All thumbnail images will be zipped and copied to the derive directory
  • PDF, HOCR & Text: The final PDF, HOCR, and text files (if they exist) will be copied
  • Main Thumbnail: The main thumbnail image (__ia_thumb.jpg) will be copied to the derive directory
  • Database Record: Job will be marked as derived in the database
  • Cleanup: Original output, upload, and thumbnail folders will be deleted to save space
  • Location: Files will be saved to media/derived/[Identifier]/
Important:
  • If a derive directory with the same Identifier already exists, it will be overwritten
  • After deriving, the job page will show the derived files instead of the output folder
  • This action cannot be undone - original files will be permanently deleted
